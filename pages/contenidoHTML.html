Modificar el contenido HTML

JavaScript nos permite modificar de forma dinámica el contenido HTML de nuestras páginas.

Por ejemplo, con JavaScript podemos seleccionar un elemento de nuestra página web que queremos que cambie de aspecto cuando el usuario haga click en un botón:

¡Cámbiame!

Para ello, podemos añadir un evento al butón que esté vigilante para ver si el usuario hace click. Podemos crear una función para que, si se hace click, entonces cambiará el texto por otro distinto. Esto se puede hacer gracias al evento onclick de JavaScript: 

          
            < button 
              type="button"
              onclick='document.getElementById("demo").innerHTML = "Hello JavaScript!"'
            >
              Haz click para cambiar el texto de al lado
            < /button >
