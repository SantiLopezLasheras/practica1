Modificar estilos CSS

Además de modificar el contenido HTML y sus atributos, JavaScript también nos permite modificar de forma dinámica los estilos CSS de nuestra página.

Por ejemplo, imagina que queremos que cuando el usuario haga click en un botón, cambie el color del texto:

¡Cámbiame!

Para ello, podemos añadir un evento al butón para que, si se hace click, el texto de al lado cambie el valor de su propiedad color a blanco. Esto se puede hacer gracias al evento onclick de JavaScript: 

          
            < button 
              type="button"
              onclick='document.getElementById("changeMe").style.color="#eee"'
            >
              Haz click para cambiar color del texto de al lado
            < /button >
